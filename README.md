# inatel-ic-pressure-monitor

## 📖 Project Description

This project was developed as part of an **Undergraduate Research Project (Iniciação Científica)** at **Inatel – Instituto Nacional de Telecomunicações**.

The main objective of this project is to develop a **pressure monitoring system** using an **ESP32 microcontroller** and **FSR (Force Sensitive Resistor) sensors**.

The system reads pressure values from the sensors and processes the data for monitoring and analysis. The ESP32 reads the analog signal generated by the sensor using its ADC (Analog-to-Digital Converter) and sends the data for processing and visualization.

This project focuses on the integration between **hardware and software**, combining embedded programming with sensor-based data acquisition.

---

## 🎯 Objectives

* Develop a system capable of reading pressure values using **FSR sensors**
* Use the **ESP32 ADC** to convert analog signals into digital values

## 🔌 Circuit Description

The circuit uses a **voltage divider configuration**.

The FSR sensor changes its resistance depending on the pressure applied to it. This variation in resistance generates a corresponding voltage change.

This voltage is read by the **ADC pin of the ESP32**, allowing the microcontroller to measure the applied pressure.

Basic configuration:

3.3V
│
FSR Sensor
│
ADC Pin (ESP32)
│
Resistor
│
GND

This configuration allows the ESP32 to measure pressure variations applied to the sensor.

---

## 💻 System Operation

1. The **FSR sensor detects applied pressure**
2. The **sensor resistance changes according to the applied force**
3. The **voltage divider generates a proportional voltage**
4. The **ESP32 reads the voltage using its ADC**
5. The data can be sent to a computer through **serial communication**

## 📊 Example Data Output

Example of data collected by the system:

FRAME,T=0,ALERTA=0,PMAX=12
FRAME,T=200,ALERTA=0,PMAX=24
FRAME,T=400,ALERTA=1,PMAX=37

Where:

* **T** → time
* **PMAX** → maximum detected pressure
* **ALERTA** → pressure alert indicator
